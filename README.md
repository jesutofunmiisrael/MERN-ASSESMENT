# MERN User Management System

A responsive full-stack MERN application for managing users.

## Features

- Create User
- Edit User
- Delete User
- View User Details
- Search Users
- Pagination
- Export CSV
- Responsive UI

## Tech Stack

### Frontend
- React
- Material UI
- React Router DOM
- Axios
- React Hook Form
- Yup Validation

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Run Frontend

```bash
cd client
npm install
npm run dev
